# Genre Playbooks

이 문서는 6개 핵심 장르를 빠르게 구분하고, 장르에 맞는 구조·톤·근거 사용 방식을 적용하기 위한 참조 가이드다.

## 1. 에세이

### 핵심 약속
개인 경험이나 관찰을 통해 독자에게 감정적 공감이나 작은 통찰을 준다.

### 좋은 시작
- 장면으로 시작
- 질문으로 시작
- 작은 균열이나 불편함으로 시작

### 기본 구조
1. 시작 장면 또는 문제의식
2. 경험/관찰
3. 그 경험이 던진 질문
4. 해석 또는 통찰
5. 여운 있는 마무리

### 어조
- 자연스럽고 과장 없음
- 감정은 구체적으로
- 교훈조는 피함

### 근거 사용
경험 중심이지만, 일반화가 커지면 사례나 자료를 보강한다.

### 먼저 묻기 좋은 질문
- 어떤 장면이 중심인가요?
- 이 경험에서 꺼내고 싶은 의미는 무엇인가요?

### 흔한 실패
- 일기처럼 흘러감
- 통찰 없이 감정만 많음
- 도입이 너무 길고 산만함

## 2. 칼럼

### 핵심 약속
한 가지 분명한 관점을 세우고, 독자가 그 관점의 필요성을 납득하게 만든다.

### 좋은 시작
- 최근 사례
- 흔한 통념 반박
- 독자의 불편한 질문 제시

### 기본 구조
1. 문제 제기
2. 핵심 주장
3. 근거 2~3개
4. 반론 또는 오해 처리
5. 결론과 함의

### 어조
- 선명하고 단호함
- 과격한 단정은 피함
- 독자의 사고를 밀어주는 톤

### 근거 사용
사례, 수치, 기사, 공식 자료, 경험을 균형 있게 사용한다.

### 먼저 묻기 좋은 질문
- 한 줄 주장으로 바꾸면 무엇인가요?
- 반대 입장은 무엇인가요?

### 흔한 실패
- 의견만 있고 근거가 없음
- 목소리는 강한데 구조가 약함
- 도덕 교훈조로 끝남

## 3. 기사

### 핵심 약속
사실을 검증해 맥락과 함께 명확하게 전달한다.

### 좋은 시작
- 가장 중요한 사실 먼저
- 리드 문장에서 누가/무엇을/왜 중요한지 제시

### 기본 구조
1. 핵심 사실 리드
2. 핵심 맥락
3. 배경 정보
4. 인용/데이터/공식 입장
5. 남는 질문 또는 의미

### 어조
- 감정 과잉 배제
- 사실과 해석 구분
- 검증 중심

### 근거 사용
공식 발표, 인터뷰, 보고서, 원자료를 우선한다.

### 먼저 묻기 좋은 질문
- 뉴스형인가요, 해설형인가요, 특징 기사형인가요?
- 확인 가능한 출처가 있나요?

### 흔한 실패
- 칼럼처럼 주장해버림
- 출처가 모호함
- 리드에 핵심 사실이 없음

## 4. 블로그

### 핵심 약속
독자가 원하는 정보나 경험을 쉽게, 빨리, 실제로 써먹을 수 있게 전달한다.

### 좋은 시작
- 독자의 문제 제시
- 핵심 요약
- 이 글에서 얻어갈 것 명시

### 기본 구조
1. 문제/관심사
2. 요약
3. 섹션별 본문
4. 예시/팁/경험
5. 정리와 다음 행동

### 어조
- 친근하고 이해하기 쉽게
- 문단 짧게
- 소제목 적극 활용

### 근거 사용
후기성 글이라도 가격, 날짜, 성능, 수치 등은 검증한다.

### 먼저 묻기 좋은 질문
- 독자가 무엇을 검색해서 들어오나요?
- 실용 팁이 필요한가요, 후기 중심인가요?

### 흔한 실패
- 제목과 본문이 다름
- 감상만 많고 정보가 없음
- 검색 의도를 못 맞춤

## 5. 뉴스레터

### 핵심 약속
이번 호만의 관점을 주고, 독자가 다음 호도 기다리게 만든다.

### 좋은 시작
- 이번 호 한 줄 메시지
- 개인적 관찰 + 지금 중요한 이유
- 짧은 훅

### 기본 구조
1. 오프닝
2. 이번 호 핵심 요약
3. 메인 섹션 1~3
4. 실용 포인트/인사이트
5. 클로징

### 어조
- 친근하지만 느슨하지 않게
- 반복 발행을 견디는 안정적 리듬
- 독자와 대화하듯

### 근거 사용
큐레이션과 해설의 균형을 맞춘다.
링크나 원자료를 줄 수 있으면 더 좋다.

### 먼저 묻기 좋은 질문
- 발행 주기는 어떤가요?
- 이번 호의 핵심 메시지는 무엇인가요?

### 흔한 실패
- 정보 나열만 하고 해석이 없음
- 섹션이 너무 많아 산만함
- 매 호 톤이 흔들림

## 6. 보고서

### 핵심 약속
상대가 빨리 이해하고 판단하게 만드는 문서를 만든다.

### 좋은 시작
- 핵심 요약부터
- 무엇을 결정해야 하는지 먼저

### 기본 구조
1. Executive Summary
2. 배경/목적
3. 분석 기준 또는 방법
4. 핵심 결과
5. 시사점
6. 제안/다음 조치

### 어조
- 단정하고 효율적
- 메시지 중심
- 제목과 소제목이 내용 요약 역할을 하게

### 근거 사용
수치, 기준, 시점, 비교, 출처를 분명히 한다.

### 먼저 묻기 좋은 질문
- 보고 대상은 누구인가요?
- 무엇을 판단해야 하나요?
- 조직 양식이 있나요?

### 흔한 실패
- 결론이 뒤로 밀림
- 핵심 메시지가 문단에 묻힘
- 숫자는 많은데 의미 설명이 없음

## 장르가 애매할 때의 판단 기준
- 개인 경험과 통찰 → 에세이
- 관점과 주장 → 칼럼
- 사실과 맥락 전달 → 기사
- 검색성과 실용 정보 → 블로그
- 반복 관계형 콘텐츠 → 뉴스레터
- 판단과 실행 전달 → 보고서

## 장르 선택 후 바로 해야 할 일
1. 독자 수준 확정
2. 구조 골격 확정
3. 필요한 근거 종류 확정
4. 개요 먼저 작성
